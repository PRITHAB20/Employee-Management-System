Employee Management System

Project Overview

The Employee Management System is a web-based application developed using Advanced Java technologies. The system allows administrators to manage employee records and employees to view their profiles through a secure login system.

This project demonstrates the implementation of JSP, Servlets, JDBC, MySQL Database Connectivity, JavaMail API, and Apache Tomcat Deployment.

---

Technologies Used

- Java 8
- JSP
- Servlets
- JDBC
- MySQL 8.0
- Apache Tomcat 9
- HTML & CSS
- JavaMail API
- Eclipse IDE

---

Features

Login Authentication

- Secure login using username and password
- Role-based access (Admin and Employee)

Employee Management

- Add Employee
- View Employee
- Update Employee
- Delete Employee

Pagination and Sorting

- Display employee records page-wise
- Sort records by:
  - Name
  - Department
  - Salary

Email Notification

- Email sent when employee records are created or updated

Deployment

- Deployable on Apache Tomcat Server
- WAR file generation supported
